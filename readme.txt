### 📁 Estrutura do Projeto `chatsurtado4.7AI`

```
chatsurtado4.7AI/
├── julia.py
├── china.py
├── ravena.py
├── app_desbugaxuxu_x.py
├── audio/
│   ├── fala_julia_*.wav
│   ├── fala_china_*.wav
│   └── fala_ravena_*.wav
├── readme.txt
├── requisitos.txt
├── assets/
│   └── logo.png
```

### 🔧 Requisitos

- Python 3.x
- pyttsx3
- datetime (built-in)

Para instalar o pyttsx3:
`pip install pyttsx3`

### 🛠️ Como rodar

Para rodar cada script e gerar os áudios:

`python julia.py`
`python china.py`
`python ravena.py`
`python app_desbugaxuxu_x.py`

Os arquivos de áudio serão salvos na mesma pasta dos scripts. Você pode movê-los para a pasta `audio/` manualmente ou adaptar os scripts para salvar diretamente lá.
